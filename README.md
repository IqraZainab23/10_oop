# OOP Project

## Overview
This is a simple Number Guessing game implemented in Typescript.

## Usage
Run the OOP:
```bash
npx 10_oop
```

## Dependencies
`Inquirer`: Provides the interactive prompt interface.

`Chalk`: For styling and coloring the output.
